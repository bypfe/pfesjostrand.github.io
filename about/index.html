About

Filip Sjöstrand / 2018-12-02

About me. I am another guy from Sweden. Online I generally go by PFE or some variation of that ; based on my full name Per Filip Emil Sjöstrand. I split my time between Gothenburg, Lund and Stockholm. Contact me on pfesjostrand@gmail.com, for my resume see LinkedIn. My education is in political science (and some economics and statistics). I also have extensive practical experience in politics from volunteering as a local organizer for Socialdemokraterna, Sweden’s social-democratic party. While I am no longer active in any party, politics in different ways have been an interest more or less as far back as I remember and I’d like to keep working with it or related fields in some way.

About site. This website was constructed using Hugo and blogdown, an R package built by Yihu Xie, which main benefit is that R code can be written in the contents as in the example below. Pretty awesome if you want to present data analysis.

    c(greeting)
